# Law Firm Website (Full-Stack Web Application)

Overview:

This project is a dynamic law firm website developed using PHP, HTML, CSS, and JavaScript. It provides a professional platform for legal services, allowing users to explore firm information, services, and contact options through an interactive web interface.

The goal of this project is to demonstrate full-stack web development skills by building a functional and responsive business website.

Key Features:
Responsive and user-friendly design,
Dynamic content using PHP,
Interactive frontend using JavaScript,
Service and information pages,
Contact form functionality,
Structured navigation system.

System Design:
Frontend,
HTML – structure of the website,
CSS – styling and layout,
JavaScript – interactivity and dynamic behavior.

Backend:
PHP – server-side logic and dynamic content rendering,
Handles form submissions and page interactions.

Technologies Used
HTML,
CSS,
JavaScript,
PHP.
